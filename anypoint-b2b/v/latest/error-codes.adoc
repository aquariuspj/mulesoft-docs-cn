== 添加新的错误代码

. 在<<img-error-codes>>上点击*New*。
+
出现<<img-error-code>>。

[[img-error-code]]

image::error-codes.png[img-error-code, title="错误代码页"]

[start=2]

. 输入所需的设置（请参阅<<Error Code Settings>>）。 +
. 输入与此合作伙伴相关的可选设置。
. 点击*Save*保存设置。

== 编辑现有的错误代码

. 在<<img-error-codes>>上的*Code*列中，点击您要修改的代码。
. 编辑所需的设置（请参阅<<Error Code Settings>>）。 +
. 编辑与此合作伙伴相关的可选设置。
. 点击*Save*保存设置。
