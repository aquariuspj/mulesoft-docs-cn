= 工作流程：使用API​​功能监控测试API

您可以测试任何REST API，例如部署在Internet上的服务器上的Deck of Cards API。例如，BAT演示github项目使用卡片卡API。

此工作流程涵盖了从创建到测试和简化API测试报告的所有内容。在Anypoint Platform上创建并部署API之后，您可以创建一个简单的Hello World测试套件。
