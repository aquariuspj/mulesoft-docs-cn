= 查看和删除API警报
：imagesdir：./_images

在主组织的组织管理员角色中，您可以查看警报列表。

*Note:*要查看和删除API警报，用户必须具有“管理警报”权限。只能查看
警报，用户必须具有查看警报权限。
请参阅 link:/api-manager/v/2.x/environment-permission-task[设置权限]。

本过程介绍如何查看和删除警报。

. 在API管理器> API管理中，单击API实例，然后选择*Alerts*。
+
image::new-alerts.png[宽度= 687，高度= 235]
+
. 在列表中展开提醒。
+
有关警报选项的详细信息出现。
+
. 点击删除。
