= 添加API提醒

. 在API Manager 2.x的API管理中，选择一个API实例，并在API实例仪表板上选择警报。
. 点击添加提醒并设置提醒选项：
* 名称：输入警报名称，例如`red-alert`。
* 启用：例如，接受默认的“启用”。
* 严重性：接受默认的Critical。
* 警报类型：例如，选择请求计数。
* 发生的次数是：选择更大。
* 比：选择1。
* 至少：选择范围为1  -  999999999的条目以将时间间隔指定为滑动窗口。
* 收件人：选择您的业务组中用户的名称，或者键入电子邮件地址，用户将在警报触发时接收通知。按RETURN键。
+
. 创建提醒。
