= 管理API访问请求
:keywords: portal, api, console, documentation


以下过程假定应用程序请求从其门户访问API的Premium SLA层。

. 在API版本详细信息页面上，单击客户端应用程序，单击请求的SLA层下面的SLA层的名称。例如，点击Premium。
+
. 选择您想要应用到应用程序的SLA层。
. 点击批准或拒绝以授予或拒绝访问权限。
+
撤销会出现在用户界面中，取代批准，拒绝和删除。
