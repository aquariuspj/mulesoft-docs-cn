= 从群集中删除服务器（混合）

删除群集时，请考虑以下事项：

* 删除群集的警报定义和历史记录。
* 分配给群集的服务器将重新启动。
* 部署到群集中服务器的应用程序将被删除。

. 在运行时管理器中，单击服务器选项卡。
. 单击群集的名称。
. 单击群集状态旁边的下拉箭头：

+
image:cluster-delete.png[删除群集]
+
出现提示，要求您确认是否要删除群集。

. 点击复选框进行验证，然后点击删除群集。

删除集群后，每台服务器都重新启动。
