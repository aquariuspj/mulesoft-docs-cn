[TIP]
本文档涉及调用 link:/runtime-manager/cloudhub-api[CloudHub API]。有关如何通过Runtime Manager UI更轻松地执行这些相同任务的说明，请参阅 link:/runtime-manager/cloudhub[CloudHub]及其子页面。

通过/ api / notifications资源指定的POST操作会创建一个新的CloudHub通知。

*Example Request:*

[source,json, linenums]
----
POST https://anypoint.mulesoft.com/cloudhub/api/notifications
Content-Type: application/json
 
{
  "domain": "myapp",
  "priority": "INFO",
  "message": "Something happened",
}
----

响应示例：

[source,json, linenums]
----
201
Content-Type: application/json
Location: https://anypoint.mulesoft.com/cloudhub/api/notifications/12345
----
