= 设置断点

* 右键单击构建块，然后选择“切换断点”。

Studio将一个红点应用于画布上构建块的图标。

当您以调试模式运行您的应用程序时，Studio会在您设置的断点处停止流程执行，从而允许您在Mule Debugger视图中检查Mule事件内容。
