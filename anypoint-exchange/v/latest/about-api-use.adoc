= 关于消费API

Anypoint Exchange为API提供商提供了重要的功能，可以轻松地向组织内部和外部的开发人员发布和共享API。

Exchange帮助API消费者了解API功能和测试功能：

*  API控制台来查看API函数。该控制台显示在REST API详细信息屏幕的右侧。
*  API Notebook查看API文档并测试API功能。
* 使用模拟服务将数据模拟到API。
