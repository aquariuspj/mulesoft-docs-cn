=  Oracle EBS 12.1连接器发行说明
:keywords: release notes, oracle, ebs, e-business suite, connector

_Premium_

*November 2017*

用于Oracle电子商务套件的Anypoint连接器提供实时连接，以使用Web服务查询和更新Oracle EBS。

Oracle EBS连接器需要企业许可证。

- 支持SuiteTalk API 2017.1版。
- 增加了对Mule 4的支持。
