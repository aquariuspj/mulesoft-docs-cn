=  Neo4j连接器发行说明
:keywords: release notes, connector, neo4j

*January 2018*

_选择_

Neo4j的Anypoint Connector提供了执行CQL语句和运行基本操作（如创建，更新，删除和选择Neo4j Graph数据库实例的节点）的功能。
