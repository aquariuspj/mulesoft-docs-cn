== 已修复问题

此版本包含以下问题修复程序：

* 将Jackson库升级到2.9.5版以修复漏洞。
* 将Jetty升级到版本9.4.8.v20171121来修复漏洞。
* 修复跟踪通知消耗并发送到数据库时的错误。
