=  Mule ESB 3.0.1发行说明
:keywords: release notes, esb


Mule团队请宣布Mule ESB 3.0.1的可用性。这是Mule 3.0.0的后续版本，它继续了MuleSoft的承诺，使Mule成为业界功能最强大，最简单易用且最新的开源ESB。

== 当前版本

|===

强调：

* 更新并重新组织在线文档
* 改进了Mule模式文档
* 现在完全支持异步流程
* 新的路由消息处理器
* 支持jBPM 4.4
* 新的BPM消息处理器/组件
* 超过40个重要修复

本版本中已修复==
