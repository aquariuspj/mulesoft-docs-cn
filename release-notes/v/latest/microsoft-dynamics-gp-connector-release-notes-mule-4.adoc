{Microsoft} GP连接器发行说明= 

*May 2018*

_选择_

Microsoft Dynamics GP连接器使Mule应用程序能够与Microsoft Dynamics Great Plains（GP）Web服务进行交互。

==  2.0.0

*May 9, 2018*

=== 兼容性

| MS动态GP
一个|  * 2010 R2
*  2013（版本12）
*  2013 R2
*  2015（版本14）
*  2015 R2
|===

=== 特点

* 增加了对Mule 4的支持。
* 仅支持Kerberos身份验证方法。
