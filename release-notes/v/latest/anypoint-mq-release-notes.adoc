=  Anypoint MQ版本说明
:keywords: mq, release, notes

Anypoint MQ是一种多租户云消息服务，它使客户能够在其应用程序之间执行高级异步消息传递方案。 Anypoint MQ与Anypoint Platform完全集成，提供基于角色的访问控制，客户端应用程序管理和连接器。
