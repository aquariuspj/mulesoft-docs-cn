== 版本1.0.0

新版本的SharePoint Online版本。

支持的操作：

* 列出和列出项目API：创建，删除，检索，更新列表和列出项目。
* 文件和文件夹API：
** 文件：添加，检索内容，检索元数据，删除，签出，撤消签出，签入，发布，取消发布，批准，拒绝，回收和复制文件。
** 文件夹：创建，删除和检索文件夹。
