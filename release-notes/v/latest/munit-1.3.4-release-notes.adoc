=  MUnit 1.3.4发行说明
:keywords: munit, 1.3.4, release notes

我们很高兴地宣布Mirit版本1.3.4的新版本，其中包括几个错误修复。

此版本包含以下工件：

*  Mule MUnit支持：
**  3.7.7
**  3.8.4

*  Munit
**  1.3.4

*  MUnit Studio插件
**  1.5.1

== 硬件和软件要求

*  Microsoft Windows 8 +
*  Apple Mac OS X 10.10 +
*  Linux（在Ubuntu 15上测试）
*  Java 7，Java 8
*  Maven 3.1+

*  MUSP-432  - 类路径包含xercesImpl的冲突版本
