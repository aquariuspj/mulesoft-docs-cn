用于SOAP 1.0.4发行说明的=  APIkit

此版本的SOAP for SOAP仅包含对运行时的一个修补程序。

对于Anypoint Studio，此版本作为Anypoint APIkit SOAP Extension 1.1.4分发。

== 兼容性

[%header%autowidth.spread]
|===
| {软件{1}}版本
| Mule Runtime  | 3.7.3，3.8.x和3.9.x
| Anypoint Studio  | 6.2.3及更高版本
|===

本版本中已修复==
