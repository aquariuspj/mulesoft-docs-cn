*November 18, 2017*

=== 特点

* 此版本为更简单的映射添加了一些操作的输入变量的优化。
* 此版本与Mule 4.0兼容。
* 增加了在操作中自动完成收集名称字段的功能。
* 为文档和文件添加了元数据。
