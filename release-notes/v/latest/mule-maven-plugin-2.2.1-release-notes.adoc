=  Mule Maven插件2.2.1发行说明

MuleSoft很高兴宣布发布Mule Maven Plugin 2.2.1。此版本修复了一些JDK的兼容性问题。

== 硬件和软件要求
JRE 1.7.0（推荐的JRE 1.7.0_79 / 80），JRE 1.8，Maven 3.3.x

== 兼容性
* 核心服务API 1.0
{Restart} *  ARM Rest API 1.1
*  CloudHub API 1.0
* 运行时管理器代理API 1.0
*  Mule Kernel，Mule 3.x
