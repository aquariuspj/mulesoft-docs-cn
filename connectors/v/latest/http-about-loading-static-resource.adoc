= 关于加载静态资源
:keywords: connectors, http, https

您可以配置HTTP连接器以加载静态资源，例如您的主页或脚本。您可以设置要加载的资源的路径和文件名。您可以选择将另一个文件的名称与资源所在的目录设置为相同的目录，以便在发生故障时使用。
