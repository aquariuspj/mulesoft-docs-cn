=  NetSuite连接器参考

+++
NetSuite连接器同步数据并自动化NetSuite和第三方应用程序之间的业务流程，无论是内部部署还是云中。 Connector使用SuiteTalk WSDL提供基于SOAP的集成，以便生成NetSuite业务对象，利用不同的身份验证级别并支持错误处理。
+++
