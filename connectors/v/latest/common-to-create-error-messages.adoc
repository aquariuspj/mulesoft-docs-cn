// Anypoint Studio，Design Center：FTP连接器

允许您将自定义错误消息映射到操作可以返回的错误，例如CONNECTIVITY或REDELIVERY_EXHAUSTED。

. 从流程中的操作的错误映射选项卡（例如删除）中，单击加号图标（+）打开操作的错误类型列表。
. 单击一个或多个错误类型。
. 为错误类型输入您的自定义错误消息。
