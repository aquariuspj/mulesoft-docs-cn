= 电子邮件模块文档参考

+++
电子邮件连接器，用于列出和发送电子邮件并在不同的邮箱中执行操作，例如删除和标记为已读。 <p>此连接器支持SMTP，SMTPS，IMAP，IMAPS，POP3和POP3s协议。
+++
