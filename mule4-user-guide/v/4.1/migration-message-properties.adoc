= 迁移消息属性

在Mule 3中，有几个属性范围可以用于不同的目的。在Mule 4中，大部分范围已被删除。

== 调用属性

现在在Mule 4中调用属性被称为`variables`。它们的行为与Mule 3中的行为完全相同。

== 入站属性，出站属性，入站和出站附件

这些范围在Mule 4中不存在。有关迁移的更多信息，请参阅 link:intro-mule-message[介绍骡子消息]。
