= 运行时（dw :: Runtime）


这里描述的功能包含在运行时模块中。该模块包含在Mule Runtime中，但您必须将标题`import dw::Runtime`添加到您的标题中，以将其导入到您的DataWeave代码中。

该模块包含允许您与DataWeave引擎交互的功能。
