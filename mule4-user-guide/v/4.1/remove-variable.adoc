= 关于移除变量
:keywords: anypoint studio, studio, mule, variable transformer, variables, set variable, edit variable, remove variable

Remove Variable组件接受一个变量名称并从Mule事件中移除该变量。

[%header,cols="30a,70a"]
|===
| {字段{1}}说明

| 显示名称（`doc:name`）
| 自定义以在您的流程中显示此组件的唯一名称。默认为Set Variable。

| 姓名（`variableName`）
| 要删除的变量的名称。
|===
