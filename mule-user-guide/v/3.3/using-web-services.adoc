= 使用Web服务

Mule旨在通过以下方式促进Web服务的使用：

可以托管或使用*  Web服务
* 传输与协议分离，也就是说，您可以通过JMS，电子邮件等进行发送/接收。
*  JAX-WS或简单服务
*  Web服务可以在不需要太多代码的情况下进行代理或修改
