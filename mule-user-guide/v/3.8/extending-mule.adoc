== 构建自定义功能

如果您发现您的集成开发团队经常需要开箱即用的Mule功能，您可以使用广泛的 link:/anypoint-connector-devkit/v/3.8[Anypoint连接器DevKit]来构建自己的连接器。

使用DevKit来：

* 构建您自己的*connectors*以在Mule应用程序中使用
* 构建您可以添加到Studio实例的*modules*
* 将现有的Mule元素扩展为*customize*，以满足您的需求
