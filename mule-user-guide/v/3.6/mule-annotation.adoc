=  @ Mule注解
:keywords: annotations, custom java code

参数注入注释可用于组件入口点和转换器方法，此注释可用于对消息负载执行 link:/mule-user-guide/v/3.6/mule-expression-language-mel[骡子表达]，并将结果传递到方法中。

[source, java, linenums]
----
public class MyComponent {
    public Object process(@Mule("message.payload") String payload) {
        // do stuff
    }
}
----

Mule定义了访问当前消息和Mule环境的微语言，更多信息可以是 link:/mule-user-guide/v/3.6/mule-expression-language-mel[在这里找到]。
