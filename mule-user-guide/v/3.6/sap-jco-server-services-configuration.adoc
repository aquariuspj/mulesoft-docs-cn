=  SAP JCo服务器服务配置
:keywords: anypoint studio, esb, connector, endpoint, sap

如果您想要接收IDoc或被称为BAPI，并且您不希望将PORT编号用作*jcoGwService*的值，那么您将不得不修改操作系统`services`文件：

* 基于Unix的系统：`/etc/services`
*  Windows：`C:\WINDOWS\system32\drivers\etc\services`

[NOTE]
====
JCo需要此配置才能启动需要与SAP网关通信的JCo服务器。

您可以在TCP端口sapgw \ {nr}下的每个应用程序服务器上访问SAP网关，其中\ {nr}是应用程序实例的实例编号。

您需要在`services`文件中添加该SAP网关条目。因此，例如，如果您尝试访问您的SAP实例21，那么您需要添加：

[source, code, linenums]
----
sapgw21  3321/tcp
----

有关更多信息，请检查您的SAP文档。
