=  Mule管理控制台文档

Mule ESB管理控制台集中管理所有Mule企业版部署的管理，无论是单独的还是嵌入在应用程序服务器中的。

如果您准备好直接进入，请直接进入 link:/mule-management-console/v/3.3/quick-start-guide-to-mule-esb-server-and-the-management-console[快速入门指南]。
