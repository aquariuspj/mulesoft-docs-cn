=  Mule管理控制台文档

Mule ESB的管理控制台提供了一种集中的方式来管理所有的Mule企业版部署，既可以独立使用，也可以嵌入应用服务器中。

准备好了吗？如果是这样，请直接转到 link:/mule-management-console/v/3.2/quick-start-guide-to-mule-esb-server-and-the-management-console[快速入门指南]。
