=  MMC REST API

管理控制台提供了REST API，您可以通过它编程访问控制台的大部分功能。使用这些REST API，您可以：

* 注册新服务器并管理现有服务器，包括重新启动服务器。
* 管理服务器组。
* 访问和管理服务器上的文件。
