= 退出可视化设计模式

_Requires November 2017 Release_

当您单击编辑RAML时，您将离开视觉设计模式并且不能返回。您随后可以在代码视图中修改您的API规范。如果您不_单击编辑RAML，您可以随时在视觉设计模式下继续处理您的API规范。

. 点击编辑RAML。
+
image::raml-try-it-tabs.png[高度= 103，宽度= 177]
+
. 同意离开视觉设计模式。
+
您创建的API规范出现在代码视图中。
+
. 在代码视图中继续您的工作。
