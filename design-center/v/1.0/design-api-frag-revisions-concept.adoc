= 关于API片段

_Requires November 2017 Release_

API片段是具有版本和标识符的RAML文档，但本身并不是完整的RAML规范。 API片段是由RAML.org定义的以下类型之一：

* 性状
* 资源类型
* 库
* 输入
* 用户文档
* 示例
* 注释类型
* 安全方案

您可以将API片段发布到Exchange，将片段作为依赖项包含在项目中，然后修改片段。修改片段后，更新使用片段的API规范以包含新版本。

要使任何API规范或片段可供其他人使用，您需要将设计发布到Exchange。
