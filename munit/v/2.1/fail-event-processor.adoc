= 关于失败事件处理器
：版本信息：2.0及更高版本
:keywords: mule, esb, tests, qa, quality assurance, verify, functional testing, unit testing, stress testing

失败事件处理器允许您故意失败测试。这对验证不应发生特定事件很有用。

[source,xml,linenums]
----
<munit-tools:fail
  message="This should fail"/>
----

消息字段是可选的，并定义断言失败时要打印的错误消息。
