= 关于MUnit Maven插件
：版本信息：2.0和更新
:keywords: munit, testing, unit testing

MUnit是由几个模块组成的，所有这些模块都是运行MUnit测试所必需的。

MUnit提供了一个Maven插件，它允许您将MUnit测试作为持续集成环境的一部分运行。 +
MUnit Maven插件允许您从Maven运行MUnit测试。
